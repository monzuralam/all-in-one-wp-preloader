== Description ==

= All-in-One WP Preloader =

All-in-One WP Preloader gives your site a loading screen without writing any code.

It’s great and lightweight. Simply activate the plugin and enable the Preloader from the “All-in-One WP Preloader” menu tab.

= Works everywhere =

All-in-One WP Preloader on all major browsers and you can use it on any theme and any device.


= Features of All-in-One WP Preloader =

* Quick to set up
* Pure CSS3 animation. Works in any modern browser.
* Option to enable/disable the preloader.
* Set a custom image
* Background Color
* 3 design style
* Responsive
* Easy to use


== Installation ==

1. Install All-in-One WP Preloader by uploading the `all-in-one-wp-preloader` zip file to the `/wp-content/plugins/` directory. 
2. Activate All-in-One WP Preloader through the `Plugins` menu in WordPress.
